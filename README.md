# Simple To-Do List App

A beautiful, intuitive to-do list web app built with HTML, CSS, and JavaScript.

## Features

- ✅ **Add tasks** - Type a task and click "Add Task" or press Enter
- ✅ **Mark tasks complete** - Click the checkbox to mark a task as done
- ✅ **Delete tasks** - Remove tasks you no longer need
- ✅ **Save automatically** - Your tasks are saved in your browser and persist between sessions
- ✅ **Clean design** - Simple and intuitive interface that's easy to use

## How to Use

1. Open `index.html` in your web browser
2. Type your task in the input field
3. Click "Add Task" or press Enter to add it to your list
4. Check the checkbox next to a task to mark it as complete
5. Click "Delete" to remove a task
